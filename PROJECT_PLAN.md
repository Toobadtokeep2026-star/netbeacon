# NetBeacon

## Mission
Native iOS network and device discovery utility.

## Architecture
- SwiftUI
- MVVM
- Network.framework
- CoreBluetooth
- Bonjour/mDNS
- SwiftData
- Swift Charts

## Build phases
1. Project scaffold
2. Local network discovery
3. BLE discovery
4. AirPlay and Cast discovery
5. Device history
6. Dashboard
7. Testing and sideload preparation

## Constraints
- Free development
- iPhone 14 target
- No backend required
- Public Apple APIs only
