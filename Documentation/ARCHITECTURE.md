# NetBeacon Architecture

## Overview
NetBeacon is a native iOS discovery and monitoring utility.

## Core Technologies
- SwiftUI interface
- MVVM design
- Network.framework
- CoreBluetooth
- Bonjour/mDNS
- SwiftData
- Swift Charts

## Features

### Network Discovery
Discovers local Bonjour and DNS-SD services.

### Bluetooth
Scans BLE advertisements and tracks RSSI.

### Media Discovery
Detects compatible AirPlay and casting endpoints through public discovery protocols.

## Principles
- Local-first
- Free
- Privacy focused
- Public Apple APIs only
